# mediadoor
An archive of a simple ranking bot for a discord roleplay written in Typescript and previously hosted on Replit, made in early 2024

## Features
- Server & Channel ranking based on number of messages
- Daily channel ranking every morning at 6 Mountain Time

To circumvent Replit's unreliable (& expensive...) database system, data persistence was handed through discord messages in a channel only the bot had access to.
